A fully responsive, single-page restaurant website built with pure HTML, CSS, and JavaScript. Designed for a fine dining wood-fired restaurant based in Visakhapatnam, Andhra Pradesh.

Tech Stack

HTML5 — Semantic markup
CSS3 — Custom properties, Grid, Flexbox, animations
JavaScript (Vanilla) — Tabs, scroll effects, form handling
Google Fonts — Cormorant Garamond + Josefin Sans
Google Maps Embed — Location map

How to Run

Download index.html
Open it in any modern browser — no setup required
To deploy: upload to GitHub Pages, Netlify, or Vercel


Customization Guide
Change restaurant name: Search for Ember & Oak and replace with your name
Change prices: Find ₹ in the menu section and update values
Change location: Replace the Google Maps embed src URL with your address
Change images: Replace unsplash.com URLs with your own image URLs
Change colors: Edit the CSS variables at the top of the <style> block:
css:root {
  --cream: #f5efe6;
  --dark:  #1a1209;
  --ember: #c8490a;
  --gold:  #c9a84c;
}
